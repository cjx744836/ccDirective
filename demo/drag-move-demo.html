由v-for生成的拖拽内容
            <div v-drag-move="dragData">
                <div v-for="k in list">{{k}}</div>
            </div>
            new Vue({
                el: '.app',
                data: {
                    list: [1,2,3,4,5]
                },
                methods: {
                    dragData() {
                        return {
                            direction: 'v',
                            offsetX: 0,
                            offsetY: 0,
                            zIndex: 9,
                            activeClass: '',
                            static: false,
                            onChange: (cb, tagIndex, srcIndex) => {
                                this.list = cb(this.list);
                            }
                        }
                    }
                }
            });
            非v-for的拖拽内容，可以不提供对象或返回对象的函数而直接使用指令，会使用默认值
            <div v-drag-move="{static:true,direction:'v'}">
                <div>1</div>
                <div>2</div>
                ...
            </div>
            或
            <div v-drag-move>
                <div>1</div>
                <div>2</div>
                ...
            </div>
            new Vue({
                el: '.app',
                data: {
                    list: [1,2,3,4,5]
                },
                methods: {
                    dragData() {
                        return {
                            direction: 'v',
                            offsetX: 0,
                            offsetY: 0,
                            zIndex: 9,
                            activeClass: '',
                            static: true,
                            onChange: (tagIndex, srcIndex) => {
                                ...
                            }
                        }
                    }
                }
            });
            指令v-drag-move，接收对象或返回对象的函数，也可以只使用指令不传任何值
            

                
                    

                        属性
                        值
                        说明
                    

                
                
                    

                        direction
                        'v' | 'h' | 'vh'，默认值'v'
                        拖拽方向。'v'竖向, 'h'横向, 'vh'横竖向
                    

                    

                        activeClass
                        默认值''
                        拖拽激活时的激活css类
                    

                

                        offsetX
                        默认值0
                        横向偏移量
                    

                

                        offsetY
                        默认值0
                        竖向偏移量
                    

                    

                        zIndex
                        默认值9
                        报拽层的堆叠顺序
                    

                    

                        static
                        默认值false。当没有给指令提供函数时默认值为true。
                        非v-for生成的拖拽内容需要指定为true
                    

                

                        onChange
                        当static为true时，参数tagIndex目标索引, srcIndex源索引
当static为false时，参数cb交换函数, tagIndex目标索引, srcIndex源索引。
cb交换函数，接受一个参数即v-for绑定的数据数组，并返回一个新的交换后的数据数组		
                        位置交换成功后的回调函数。
